🍽️ Power BI Restaurant Insights Hub

A centralized Power BI analytics solution designed to monitor, compare, and optimize performance across multiple restaurants.

📊 Project Overview

This project delivers a comprehensive business intelligence solution for two restaurants by integrating their data into a unified Power BI dashboard. It enables both individual restaurant analysis and cross-restaurant performance comparison.

The solution is deployed in Power BI Service with role-based access control, ensuring secure and customized data visibility for different users.

🚀 Key Features

 📌 **Centralized Dashboard**

  * Unified view of multiple restaurants
  * Cross-performance comparison

 🏪 **Individual Restaurant Reports**

  * Dedicated reports for each restaurant
  * Focused operational insights

 📈 **Overall Analytics Report**

  * Aggregated metrics across all restaurants
  * Trend analysis and KPI tracking

 🔐 **Role-Based Access Control (RLS)**

  * Secure data access for different users
  * Customized views based on roles

🛠️ Tech Stack & Tools

 **DAX (Data Analysis Expressions)**
 **Power Query (ETL & Data Cleaning)**

1. **Data Collection**

3. **Data Modeling**

6. **Deployment**
